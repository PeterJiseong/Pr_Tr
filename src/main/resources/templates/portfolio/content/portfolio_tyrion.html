CulFoShe Project

Spring Boot 기반의 웹 애플리케이션 프로젝트

교육과정 프로젝트

프로젝트 개요

프로젝트 배경

국비지원 교육과정에서 진행한 팀 프로젝트로, Spring Boot를 활용한 웹 애플리케이션 개발을 목표로 했습니다.

프로젝트 상태

팀원들의 진로 변경으로 인해 프로젝트가 완료되지 않았으나, Spring 프레임워크와 웹 개발의 기초를 다지는 중요한 학습 경험이었습니다.

사용 기술

기술 스택

Spring Boot Thymeleaf Oracle JPA CSS JavaScript Spring Security HTML

프로젝트 의의

학습 성과

Spring Boot 프레임워크의 기본 구조와 작동 원리를 이해하고, 웹 애플리케이션 개발의 전반적인 프로세스를 경험했습니다.

기술 습득

JPA를 통한 데이터베이스 연동, Spring Security를 활용한 보안 설정, Thymeleaf를 이용한 화면 구현 등 실제 프로젝트에 필요한 핵심 기술들을 학습했습니다.
